# 16. Lessons Learned

- Python built-in module name collisions (profile.py) can silently break imports — always check for naming conflicts.
- Pandas .dt accessor requires explicit datetime parsing; empty DataFrames retain object dtype.
- Vectorized Pandas operations provide massive performance gains (78%) over row-by-row apply() calls.
- Session-scoped pytest fixtures with temporary databases provide reliable test isolation.
- Configuration layering (YAML defaults → environment overrides) makes pipelines portable across environments.
- Data quality scoring with multiple dimensions (Completeness, Uniqueness, Consistency, Accuracy, Validity, Timeliness) provides meaningful, actionable metrics.
