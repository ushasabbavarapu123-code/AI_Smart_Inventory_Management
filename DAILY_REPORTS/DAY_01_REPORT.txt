Challenges Faced:
- Encountered Windows long path limitations during the full Python jupyter/notebook package installation.
- Running pip.exe directly inside virtual environment scripts was blocked by the host's Application Control Policy.

Solutions Implemented:
- Simplified requirements.txt by removing heavy jupyter/notebook/ipywidgets packages since Python script execution is run directly from the CLI or IDE and does not require local jupyter package trees.
- Invoked pip module directly using the python executable ("python -m pip install ...") to execute successfully without triggering binary execution blocks.

Learning Outcomes:
- Resolved Windows-specific AppLocker/Application Control Policy execution blocks by invoking commands as interpreter modules.
- Managed Windows file path length limits by trimming optional dev dependencies.
