# 16. Lessons Learned

- Python scripts run as __main__ require absolute imports (not relative package imports) — use sys.path.append()
- IQR outlier method is conservative and may allow edge-case negatives depending on distribution shape; validation of lower bounds needed
- Separating extract, clean, transform, validate, export into separate modules (SRP pattern) enables isolated testing and easy debugging
- Timestamped export archival provides reproducible datasets without overwriting active files
- Feature engineering for inventory analytics (safety stock, days-to-stockout, demand category) provides powerful signals for ML model input
