# 16. Lessons Learned

- Temporal train/validation splitting is essential for time series forecasting tasks to prevent lookahead bias.
- Recursive multi-step predictions allow utilizing autoregressive models like Random Forest for arbitrary forecast horizons, though errors can propagate.
- Windows CP1252 character mapping can raise runtime errors when printing unicode characters, so ASCII fallback or explicit UTF-8 outputs should be preferred.
- Timezone-aware datetimes with `timezone.utc` should be used instead of deprecated `datetime.utcnow()`.
