Dashboard:
  GET    /api/dashboard/summary - Retrieve KPI metrics summary (JWT protected)

================================================================================
ISSUES ENCOUNTERED
================================================================================

1. Jest ES Module issue with uuid library: The default v14 uuid npm package exports
   dist modules using ESM `export` syntax, which crashed Jest under CommonJS configuration.
   Resolution: Swapped uuid library out for Node's built-in `crypto.randomUUID()`,
   which requires zero NPM packages, executes faster, and supports CommonJS out-of-the-box.

2. Forecast process spawn ENOENT: Resolving the Python interpreter binary path failed
   due to controller nesting folders structure (server.js in app/src/ versus controllers
   in app/src/controllers/).
   Resolution: Adjusted path resolution back 3 parent directories: `../../../analytics/venv/`.
